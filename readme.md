# Диктофон с графическим интерфейсом

## Python, Tkinter, PyAudio

+ Требования python3 и и библиотека pyaudio

+ Скачать приложение

+ Создание виртуального окружение для python

```cmd
python -m venv env
```

+ установка библиотеки pyaudio

```cmd
pip install pyaidio
```

+ запустите приложение

```cmd
python main.py
```
